# LegalCaseAnalyzer.java
Legal Case Analyzer is a Java-based console application that presents concise summaries of landmark court cases. The program allows users to select a case and view its core legal issue, the court’s decision, and its broader impact. The project demonstrates foundational legal reasoning, structured analysis, and basic Java programming skills.
This project is a Java console application designed to analyze and explain landmark legal cases in a clear and structured format. Users can select a case from a menu and view a breakdown of the legal issue, the court’s ruling, and the significance of the decision.

The goal of this project is to demonstrate curiosity about law and civic studies while applying object-oriented programming concepts in Java. It emphasizes clarity, logical structure, and factual presentation of legal principles.
